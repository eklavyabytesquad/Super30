Interactive CSS Selectors Demo

:nth-child Selector

The :nth-child() selector allows you to select elements based on their position within a parent element.

It can take various forms:

  • :nth-child(3) - selects the 3rd child
  • :nth-child(odd) or :nth-child(2n+1) - selects odd children (1st, 3rd, 5th, etc.)
  • :nth-child(even) or :nth-child(2n) - selects even children (2nd, 4th, 6th, etc.)
  • :nth-child(3n) - selects every third child (3rd, 6th, 9th, etc.)
  • :nth-child(3n+1) - selects every third child, starting from the first (1st, 4th, 7th, etc.)

:hover Selector

The :hover selector is triggered when the user hovers their mouse over an element.

It's commonly used for creating interactive elements like buttons, links, and menu items.

Hover Me!
.hover-box:hover { background-color: #2980b9; transform: scale(1.1); }
Try hovering your mouse over the box above to see the effect!

:active Selector

The :active selector is triggered when an element is being activated by the user (e.g., when a button is being clicked).

It's often used to provide visual feedback during the clicking action.

Click & Hold Me!
.active-box:active { background-color: #c0392b; transform: scale(0.95); }
Try clicking and holding the button to see the :active state!

::after Pseudo-element

The ::after pseudo-element creates a virtual element that appears after the content of the selected element.

It must be used with the content property to specify what to insert.

It's useful for adding decorative elements without modifying the HTML structure.

Hover Me
.after-box::after { content: "→"; position: absolute; right: -20px; } .after-box:hover::after { right: -30px; }
Notice the arrow to the right of the box. Hover to see it move!

Combined Selectors

CSS selectors can be combined to create more complex and interesting effects.

Here's an example using :nth-child, :hover, :active, and ::after together:

.menu li:nth-child(odd) { background-color: #3a536b; } .menu li:hover { background-color: #2c3e50; } .menu li:active { background-color: #1a252f; } .menu li::after { content: ""; /* empty content */ } .menu li:hover::after { width: 100%; /* reveal the underline */ }

Try hovering and clicking on the menu items to see different effects:

  • Odd-numbered items have a slightly different background (:nth-child)
  • Items change color when hovered (:hover)
  • Items darken when clicked (:active)
  • A red underline appears when hovered (::after + :hover)
