Header Section: px vs rem

Fixed Width Header (px)

This header has a fixed width of 500px and uses pixel-based measurements for font sizes and spacing.

On smaller screens, this fixed width can cause layout problems, and the text sizes won't adjust.

Responsive Header (rem)

This header uses width: 100% with a max-width in rem units, and all typography uses rem.

On smaller screens, everything scales proportionally when we adjust the root font size in the media query.
