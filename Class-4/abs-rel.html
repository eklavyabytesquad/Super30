Relative vs Absolute Positioning

Static Position (Default)

Static

Default position is static, which follows the normal document flow.

Relative Position

Relative

Position: relative moves an element relative to its normal position.

top: 20px, left: 30px

Absolute Position

Absolute

Position: absolute removes the element from the document flow and positions it relative to its nearest positioned ancestor.

top: 30px, right: 20px

Note: The parent div has position: relative to create a positioning context.

Absolute (No Positioned Parent)

Absolute

Without a positioned parent, the absolute element positions relative to the document body.

Note: This parent has no position set, so the box positions to the viewport.
