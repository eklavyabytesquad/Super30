Typography: px vs rem

Typography with Fixed Units (px)

This heading is 28px and this text is 16px. These sizes are fixed and don't adapt to device size or user preferences.

On mobile screens, this text stays exactly 16px, which might be too large for the layout or too small for readability.

Typography with Relative Units (rem)

This heading is 1.75rem and this text is 1rem. These sizes adjust relative to the root font size.

On mobile screens, we reduce the root font size to 14px, and all rem units scale down proportionally, making content fit better.
